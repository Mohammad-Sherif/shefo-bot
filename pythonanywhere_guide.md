# 🚀 دليل رفع بوت صاحبك (Shefo-Bot) على PythonAnywhere

هذا الدليل يشرح خطوة بخطوة كيفية استضافة بوت التليجرام الخاص بك على منصة [PythonAnywhere](https://www.pythonanywhere.com/) مجاناً بالكامل.

---

## ١. إنشاء الحساب
1. اذهب إلى موقع [PythonAnywhere](https://www.pythonanywhere.com/).
2. اضغط على **Pricing & signup**.
3. اختر الخطة المجانية بالضغط على **Create a Beginner account**.
4. قم بتعبئة بياناتك (Username, Email, Password) ووافق على الشروط، ثم اضغط **Register**.

---

## ٢. فتح شاشة الأوامر (Bash Console)
1. بعد تسجيل الدخول، ستظهر لك لوحة التحكم (Dashboard).
2. انزل إلى قسم **Consoles**.
3. تحت عنوان **New console:**، اضغط على كلمة **Bash**.
4. ستفتح لك شاشة سوداء تشبه شاشة الأوامر في الكمبيوتر.

---

## ٣. تحميل الكود وتثبيت المكتبات
انسخ الأوامر التالية والصقها في الشاشة السوداء بالترتيب (اضغط `Enter` بعد كل سطر وانتظر حتى ينتهي):

**أولاً: تحميل الكود من GitHub**
```bash
git clone https://github.com/Mohammad-Sherif/shefo-bot.git
cd shefo-bot
```

**ثانياً: تثبيت المكتبات اللازمة**
*(مهم جداً كتابة `--user` في نهاية الأمر حتى تنجح العملية في الحساب المجاني)*
```bash
pip install -r requirements.txt --user
```

---

## ٤. إضافة مفاتيح الـ API (الـ .env)
البوت يحتاج إلى التوكن ومفاتيح الذكاء الاصطناعي لكي يعمل.
في نفس الشاشة السوداء، اكتب الأمر التالي لإنشاء ملف الإعدادات:
```bash
nano .env
```
ستفتح لك شاشة فارغة، قم بنسخ ولصق السطور التالية داخلها (تأكد من وضع المفاتيح الخاصة بك):

```env
TELEGRAM_BOT_TOKEN=ضع_التوكن_هنا
GROQ_API_KEY=ضع_الكي_هنا
CITY=Tanta
COUNTRY=Egypt
PRAYER_METHOD=5
GROQ_MODEL=llama-3.3-70b-versatile
```

**كيف تحفظ الملف؟**
1. اضغط من لوحة المفاتيح على: `Ctrl + X`
2. سيطلب منك التأكيد، اضغط حرف `Y`
3. ثم اضغط `Enter`.

---

## ٥. تشغيل البوت للأبد
الآن حان وقت تشغيل البوت. اكتب هذا الأمر:
```bash
python bot.py
```

إذا ظهرت لك رسالة تقول `✅ البوت شغّال! ابعت /start على تليجرام`، فهذا يعني أن البوت يعمل بنجاح ويمكنك الذهاب لمراسلته!

---

## ٦. تحديث الكود في المستقبل
إذا قمت بتعديل الكود على GitHub وأردت تحديث البوت ليعمل بالنسخة الجديدة:
1. اذهب للوحة تحكم PythonAnywhere وافتح الـ **Bash** console.
2. أوقف البوت القديم بالضغط على `Ctrl + C`.
3. اكتب أوامر التحديث:
```bash
cd shefo-bot
git pull
python bot.py
```

### ⚠️ ملاحظات هامة:
* **إعادة التشغيل الدورية:** في الخطة المجانية، تقوم منصة PythonAnywhere بإعادة تشغيل السيرفرات كل بضعة أسابيع للقيام بأعمال الصيانة. إذا وجدت أن البوت توقف فجأة عن الرد، كل ما عليك فعله هو الدخول للموقع، فتح شاشة الـ Bash، وكتابة `cd shefo-bot` ثم `python bot.py` مرة أخرى.
* **البروكسي (ProxyError):** قد تلاحظ رسالة خطأ حمراء تتضمن `503 Service Unavailable` في الشاشة. هذا طبيعي جداً في الحسابات المجانية بسبب نظام الحماية (البروكسي) الخاص بالموقع. البوت مبرمج ليتجاهل هذا الخطأ ويستمر في العمل أو يقوم بإعادة المحاولة فوراً، لذا **تجاهلها تماماً**.
