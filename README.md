# 🤖 بوت صاحبك — المرافق الذكي

بوت تليجرام ذكي يعمل كصديق حقيقي يهتم بصلاتك وصحتك الجسدية والروحية. يذكّرك بالصلاة، يتابع تمارينك وتغذيتك، يسأل عن حالك، ويرد على أسئلتك بالعربية الفصحى البسيطة.

## ✨ المميزات

### 🕌 نظام الصلاة
- تذكير فوري عند كل أذان (مواقيت حقيقية من AlAdhan API)
- تذكير كل دقيقتين لو ما صليت (دقيقة واحدة للفجر)
- تشجيع + ذكر جديد بعد كل صلاة
- عتاب بحنية لو فاتتك صلاة
- ٥٥ ذكر وحديث وآية لا تتكرر

### 💪 نظام اللياقة
- خطة تمارين كاملة (COACH CLAUDE PRO V6.1)
- ٧ تمارين عشوائية (4 تراك + 3 جيم)
- حساب الماكروز الذكي (١٩ صنف طعام)
- تعديل الخطة والتذكر بالتعديلات
- تتبع التمرين والتغذية يومياً

### 🧠 ذكاء اصطناعي
- محادثة طبيعية بالعربية الفصحى عبر Groq AI
- يفهم اللغة الطبيعية ويرد بذكاء
- ذاكرة للمحادثات والخطط المعدّلة
- يتصرف كصديق حقيقي مش روبوت

### 💚 السؤال على الحال
- يسأل عنك كل فترة عشوائية (١-٣ ساعات)
- ينام الساعة ٢ الفجر ويصحّيك للفجر
- تقرير يومي قبل النوم
- نظام نقاط وإحصائيات

## 🚀 التشغيل السريع

### المتطلبات
- Python 3.10+
- Telegram Bot Token (من [@BotFather](https://t.me/BotFather))
- Groq API Key (من [console.groq.com](https://console.groq.com))

### التثبيت

```bash
git clone https://github.com/YOUR_USERNAME/shefo-bot.git
cd shefo-bot
pip install -r requirements.txt
```

### الإعدادات

أنشئ ملف `.env`:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
GROQ_API_KEY=your_groq_api_key
CITY=Tanta
COUNTRY=Egypt
PRAYER_METHOD=5
GROQ_MODEL=qwen/qwen3-32b
```

### التشغيل

```bash
python bot.py
```

ثم ابعت `/start` للبوت على تليجرام.

## 📱 الأوامر

| الأمر | الوظيفة |
|---|---|
| `/start` | بدء البوت وتسجيل الحساب |
| `/prayers` | حالة صلوات اليوم |
| `/plan` | خطة تمرين اليوم |
| `/workout` | تمرين عشوائي |
| `/stats` | الإحصائيات والنقاط |
| `/adhkar` | أذكار الصباح/المساء |

## 💬 المحادثة الطبيعية

| ما تقوله | ما يفعله البوت |
|---|---|
| "صليت" | تسجيل الصلاة + تشجيع + ذكر |
| "أكلت ٣ بيضات وعيش" | حساب الماكروز |
| "عايز تمرين عشوائي" | اختيار تمرين من ٧ بروتوكولات |
| "إيه تمرين النهارده؟" | عرض خطة اليوم |
| "خلصت تمرين" | تسجيل إتمام التمرين |
| أي سؤال آخر | رد ذكي كصديق |

```
shefo_bot/

├── data/

    └── shefo.db            # قاعدة البيانات (تُنشأ تلقائياً)
```
